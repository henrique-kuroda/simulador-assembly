# 🖥️ Simulador MIPS - Arquitetura e Organização de Computadores

Este é um projeto acadêmico desenvolvido para a disciplina de **Arquitetura e Organização de Computadores**, com o objetivo de simular a execução de instruções Assembly MIPS de forma visual e interativa.

## 📌 Objetivos do Projeto

- Ler um código Assembly MIPS (.txt)
- Interpretar instruções dos tipos R, I e J
- Simular passo a passo o avanço do clock da CPU
- Mostrar o estado atual dos **registradores**, **memória**, **PC** e **instrução atual**
- Visualizar o uso dos registradores em forma de gráfico
- Tornar a execução didática e interativa para estudo

---

## 🛠️ Tecnologias Utilizadas

- C#
- .NET Framework / Windows Forms
- Gráficos com `System.Windows.Forms.DataVisualization.Charting`
- Visual Studio
